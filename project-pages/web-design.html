Web Design

Cat Quiz

https://boltscape.github.io/catquiz/

For this project I made a quiz that tests user's knowledge of cat trivia. I love cats, and this marked the beginning of the start of somewhat of a theme.

Web Hosting on GitHub Pages:

I used GitHub Pages to host my quiz, as it is a free and user-friendly platform for hosting static websites. By creating a repository for my project and enabling GitHub Pages in the repository settings, I was able to publish my quiz online with ease. This approach allowed me to share my work with others without incurring any hosting costs.

Design Enhancements:

I made several improvements to the quiz’s design to enhance its visual appeal and user experience.

  • Color Scheme: I set the page background to AliceBlue (#f0f8ff) and applied a SlateBlue (#6a5acd) border to the main quiz container to create a new, cohesive visual theme.
  • Container Styling: I centered the quiz container and added a border, rounded corners (border-radius), and a box-shadow effect to give the interface a more defined and polished appearance.
  • Typography: I changed the base font to Verdana to enhance readability and increased the font sizes for the main title and section headers to improve the visual hierarchy.
  • Form Elements: I updated the styling of buttons, text inputs, and the selection menu to create a more consistent and modern interface, including adding a hover effect to the navigation buttons.
  • Interface Adjustments: I removed the "Cheat Mode" checkbox and the external footer link to streamline the user interface and reduce visual clutter.
  • Label Modifications: I changed labels like "Solution:" and "Your response:" to "Feedback:" and "Your Answer:" for greater clarity, and updated the navigation buttons to read "Previous" and "Next".

JavaScript Improvements:

The functionality of the quiz was enhanced with subtle JavaScript updates. The textarea for questions was made read-only to preserve the quiz’s integrity. Solutions are now displayed dynamically in a styled div instead of a text input, offering better presentation and user experience. These updates ensure the quiz is visually engaging, accessible, and user-friendly.


Cat Breed Facts Drag and Drop Quiz

https://boltscape.github.io/catfacts/

This project builds on the previous fact quiz by transforming it into an interactive drag-and-drop quiz centered around cat breeds. The goal was to create a visually appealing and engaging experience for users, especially for cat lovers, while showcasing my web design skills.

HTML Modifications:

The quiz structure was completely overhauled to reflect the new "Cat Facts" theme. I updated the page title, main heading, and instructions to immediately set the context for the user. The original content regarding learning theories was replaced with nine new list items, specifically curated into three distinct categories: "Big Cats," "Domestic Breeds," and "Cat Behaviors." I also updated the JavaScript logic, specifically the boxSizeArray and the everythingIsCorrect() function, to ensure the validation logic matched the new content and provided a thematic celebratory message.

CSS Styling Enhancements:

To create an immersive and visually appealing interface, I applied a custom design focused on clarity and theme:

  • Background and Fonts: I replaced the solid background with a thematic image of black cats, setting the attachment to "fixed" to create a professional scrolling effect. The font family was updated to Arial and Helvetica to ensure a clean, modern look, with main text colors changed to white for readability against the dark background.
  • Color Palette: I centered the main quiz container to improve focus and applied a cool-toned palette. The container features a light blue-gray background (#CAD8DB) with a dark blue dotted border (#2F455A). Section headers were styled with dark blue backgrounds to contrast sharply with the teal (#4DB6AC) draggable items.
  • UI Elements: The drop zones were given a light gray background to clearly distinguish them as targets. I also enhanced the warning text styling, making it bold, red, and underlined to ensure immediate user attention during interactions.
  • Drag-and-Drop Interactions: Each draggable element was styled with a polished finish to ensure an intuitive and seamless user experience.

Hosting on GitHub Pages:

Similar to my previous projects, I hosted this quiz on GitHub Pages. This platform is both free and easy to use, making it ideal for maintaining and sharing interactive projects like this.

Result:

These adjustments resulted in a polished, fully functional drag-and-drop quiz that tests users on their knowledge of felines. The visual redesign, combined with the logic updates, creates a cohesive user experience where the aesthetics directly support the content.
